# 📚 Lilou Books | פיסול ספרים ואמנות נייר

דף נחיתה מעוצב עבור **Lilou Books** - סטודיו לאמנות פיסול ספרים, סדנאות יצירה והזמנות אישיות בניהולה של להב ברק.

---

## ✨ על הפרויקט
האתר נבנה כדף נחיתה (Landing Page) מודרני, נגיש ומותאם לנייד, המציג את תהליך היצירה, הסיפור האישי של האמנית, גלריית עבודות ואפשרות להרשמה מהירה לסדנאות.

### 🛠 טכנולוגיות בשימוש:
* **HTML5 & CSS3** - מבנה ועיצוב.
* **Tailwind CSS** - לעיצוב מהיר, נקי ותגובתי (Responsive).
* **JavaScript (Vanilla)** - לניהול הגלריה (Lightbox), תפריט הנגישות וניווט.
* **Lucide Icons** - אייקונים וקטוריים נקיים.
* **Google Fonts** - שימוש בפונטים 'Heebo' ו-'Secular One' להתאמה מושלמת לעברית.
